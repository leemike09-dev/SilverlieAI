# 모바일 앱 개발 완료 (2026-03-24)

## 구현된 화면
- LoginScreen — 이름/이메일 로그인, 4개 언어 전환 버튼
- HomeScreen — 4개 메뉴 카드 (건강기록, 커뮤니티, AI상담, 알림)
- HealthScreen — 혈압/심박수/체중/혈당/메모 입력 + DB 저장
- AIChatScreen — Claude API 연동 실제 AI 답변
- CommunityScreen — 그룹 목록/생성/가입
- NotificationsScreen — 알림 목록/읽음 처리

## 다국어 지원
- 한국어/중국어/영어/일본어 4개 언어
- LanguageContext + translations.ts 구조
- 로그인 화면 상단 국기 버튼으로 전환

## 기술 스택
- React Native + Expo SDK 54
- TypeScript
- React Navigation (native stack)

## API 연결
- POST /users/ — 로그인 (이메일 중복 시 기존 사용자 반환)
- POST /health/records — 건강 기록 저장
- POST /ai/chat — Claude AI 상담
- GET/POST /community/ — 커뮤니티 그룹
- GET /notifications/:userId — 알림 목록
