# Silver Life AI — 건강 기기 연동 디자인 브리프

## 앱 기본 정보
- 대상: 한국 시니어 60세+
- AI 마스코트: 루미 (lavender 원 + 보라 하트)
- 배경: warm ivory (`#F1ECE4 → #FBF8F3`)
- 본문 최소 22px / 버튼 최소 높이 64px / 카드: 흰색 + 파스텔 칩

---

## 기능 목적
폰 건강 앱(애플 건강 / 삼성 헬스)과 연동 →
백그라운드에서 데이터 수집 → AI 분석 → 핵심 지표만 시니어 친화적으로 표시.
루미가 숫자가 아닌 "의미"로 해석해줌.

---

## 화면 표시 원칙

**표시 (사용자에게 보임)**
- 걸음수 + 목표 진행바
- 심박수 현재값 / 최저·최고
- 수면 총 시간 + 단계 요약
- SpO2
- 각 항목마다 루미 한 줄 해석

**비표시 (AI만 활용)**
- HRV, 상세 수면 단계, 스트레스 지수
- → 루미 대화·주간 리포트에 반영

---

## 디자인 요청 항목

1. **연동 온보딩 카드** — HealthScreen 상단, 미연결 시 표시, 연결 완료 시 사라짐
2. **애플 활동 링 카드** — Move / Exercise / Stand 3링 표시 (iOS 전용)
3. **스마트워치 데이터 카드** — 심박수 / 수면 / SpO2 각 카드 + 루미 해석
4. **미연결 상태** — 빈 카드 최소화, "연결하면 자동 측정" 안내
